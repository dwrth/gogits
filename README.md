# gogits = go + git + stats

Visualize local git activity stats in the terminal with Go

## Installation & Usage

```
go install github.com/dwrth/gogits/cmd/gogits@latest
```

```
gogits -add "path/to/folder"
```

```
gogits -email "your@email.com"
```
